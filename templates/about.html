{% blocktrans %} Prestashop-Sync Service is aimed to help merchants that use Prestashop platform in solving their everyday routines.

You can instantly evaluate all functionality and start using the service without need to register.
Start using it now and tell us what we need to do to make your life easier!

Prestashop-Sync Service provides following features at the moment:
  • Bulk quantity update for all products in your shop. Either using Prestashop-Sync web interface or with CSV file.
  • Update schedules. You can perform scheduled product quantity updates in your shop. Just make update file available online somewhere, set up file location and time interval in the service and you're done.
  • Store and manage multiple shops.
Some features that we are currently working on and soon to release:
  • Create new products with images (Done).
  • Ability to update any product attribute.
  • Ability to manage product images.
  • Bulk coupons import.
Have some annoying issue that you would like to solve? Please, contact us at admin@prestashop-sync.com. {% endblocktrans %}
